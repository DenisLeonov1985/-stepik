# PRD: Курс «Основы марксизма»

## 1. Контекст
Создание структурированного онлайн-курса на платформе Stepik для изучения марксистской теории.

## 2. Пользователи и сценарии
- **Студент-новичок**: хочет получить базовое понимание без перегруза.
- **Студент-исследователь**: ищет структуру и источники для глубокого изучения.

## 3. Основные требования
- 8 модулей с прогрессирующей сложностью.
- Стратегия релиза: v0.5 (модуль 1 + 1 урок из 2–3) → v1 (8 модулей).
- Видео не более 30 минут.
- Наличие тестов после каждого урока.
- Итоговый проект в конце курса.
- Для каждого урока (минимум): видео 15–25 минут, текст 1200–2500 слов, квиз 5–10 вопросов, 1 практическое задание, 2–5 ресурсов.
- Для каждого модуля (минимум): итоговый тест 15–25 вопросов, глоссарий 10–25 терминов.

## 4. Нефункциональные требования
- Доступность материалов 24/7 на Stepik.
- Читаемость текстов и чистота звука в видео.

## 5. Метрики и проверка гипотез
- Удержание (retention) по модулям.
- Средний балл за итоговый экзамен.
- Успешность беты: 20–30 участников, правки внесены по итогам.

## 6. Риски и неизвестные
- Сложность поддержания мотивации на длинной дистанции (8 месяцев).
- Технические ограничения Stepik для сложных интерактивов.
